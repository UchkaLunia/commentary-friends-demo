# Commentary Friends Studio — Presentation Demo

Presentation-only prototype for the Commentary Friends production workflow.

- Uses sample campaign data.
- All uploads, recording, processing and publishing actions are simulated in the browser.
- No database, accounts, API credentials or social-platform connections are included.
- The live GitHub Pages site is generated from `index.html` in the repository root.

## GitHub Pages

In repository **Settings → Pages**, choose **Deploy from a branch**, then select **main** and **/(root)**.
